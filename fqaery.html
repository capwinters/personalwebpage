A Combined Platform to Manage Network Quailty Data

Below is a demonstration of a BIS tool, that I designed and developed to be able to handle daily works of network performance measurement and reporting tasks. I was not familiar with Tableau at the time I developed this tool, and it was interesting to realize that, in fact I was designing a basic version of this useful and popular software :) (if you do not believe me, watch the video)

Some main functionality of this tool are as follows:

  1. Can connect to MS SQL, OLAP and ORACLE or load local csv
  2. run / edit / keep track of / organise queries  
  3. can analyse any dataset arising from query geographically or statistically
  4. exploits the fast GPS to grid algorithm, detials of which I give here together with multi-threading
  5. Can run SQL Server Report Builder templates and supports all the functionality that comes with it
  6. A dedsicated export module for KML which makes it possible to export and define 3D kpi representations
