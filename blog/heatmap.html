How to plot a heatmap in SQL Report Builder 2016 using Visual Basic

Microsoft® SQL Server® 2016 Report Builder is a nice free tool to be able to visualize data. I define it as poor mans Tableau :)

If you are comfortable with SQL and know a bit of scripting/programming, it can do many wonders. You can plot several data types, including geo-spatial data but one missing feature it has is heatmap.

Her is how you can make a rather simple heatmap with a bit of visual basic code in Report Builder:

  1. First step is creating a proper sql query. I will use synthetically created normal distributed data for this demo.

Assuming we have proper data, such a simple query as below will do the task:

 
declare @x_roundfactor int = 10
declare @y_roundfactor int = 2
 
 
select x_bins, y_bins, count(id) as samples
 
from(
 
select id
    , @x_roundfactor*cast(1.0*xval/@x_roundfactor as int) as x_bins
    , @y_roundfactor*cast(1.0*yval/@y_roundfactor as int) as y_bins
 
from [ookalsample].[dbo].[Book1]
 
) a
 
group by x_bins, y_bins
order by x_bins, y_bins

Now let's add some formatting to make it more understandable once we run the report and the matrix expands.

This matrix will expand if we run the report, but it will not apply coloring. We must add a function to the cells of the matrix that will color a cell according to the value of that cell.

To do so, right click on somewhere in gray area on the report and open report properties. You should see the following menu:

Paste the simple function below. Don't worry for the hexadecimal rgb color values. You can create any number of colors in any order from here.

 
Public Function getrgb( value as double, maximum as double ) As String
 
Dim rgb As String = "#ff0000"
 
        If (value/maximum >= 1) Then
            rgb = "#FF0000"
        ElseIf (value / maximum > 0.96) Then
            rgb = "#FF2A00"
        ElseIf (value / maximum > 0.92) Then
            rgb = "#FF5500"
        ElseIf (value / maximum > 0.88) Then
            rgb = "#FF7F00"
        ElseIf (value / maximum > 0.84) Then
            rgb = "#FFAA00"
        ElseIf (value / maximum > 0.80) Then
            rgb = "#FFD400"
        ElseIf (value / maximum > 0.76) Then
            rgb = "#FFFF00"
        ElseIf (value / maximum > 0.72) Then
            rgb = "#D4FF00"
        ElseIf (value / maximum > 0.68) Then
            rgb = "#AAFF00"
        ElseIf (value / maximum > 0.64) Then
            rgb = "#7FFF00"
        ElseIf (value / maximum > 0.6) Then
            rgb = "#55FF00"
        ElseIf (value / maximum > 0.56) Then
            rgb = "#2AFF00"
        ElseIf (value / maximum > 0.52) Then
            rgb = "#00FF00"
        ElseIf (value / maximum > 0.48) Then
            rgb = "#00FF2A"
        ElseIf (value / maximum > 0.44) Then
            rgb = "#00FF55"
        ElseIf (value / maximum > 0.4) Then
            rgb = "#00FF7F"
        ElseIf (value / maximum > 0.36) Then
            rgb = "#00FFAA"
        ElseIf (value / maximum > 0.32) Then
            rgb = "#00FFD4"
        ElseIf (value / maximum > 0.28) Then
            rgb = "#00FFFF"
        ElseIf (value / maximum > 0.24) Then
            rgb = "#00D4FF"
        ElseIf (value / maximum > 0.2) Then
            rgb = "#00AAFF"
        ElseIf (value / maximum > 0.16) Then
            rgb = "#007FFF"
        ElseIf (value / maximum > 0.12) Then
            rgb = "#0055FF"
        ElseIf (value / maximum > 0.08) Then
            rgb = "#002AFF"
        ElseIf (value / maximum > 0.04) Then
            rgb = "#0024FF"
        Else
            rgb = "#0000FF"
 
        End If
 
Return rgb
 
End Function
